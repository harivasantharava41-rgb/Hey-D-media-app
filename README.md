# Hey-D-media-app
A full-stack social media web application similar to Instagram and Twitter. Built with Python Flask, Django, JavaScript, HTML, CSS and MySQL. Features user login, photo/video uploads, likes, comments and follow system. Deployed using Jenkins CI/CD pipeline on AWS EC2 with Docker containerization and SonarQube code quality analysis.

## 📌 Project Overview
Heyd Media is a full-stack social media web application
similar to Instagram and Twitter. Built with Python Flask,
JavaScript, HTML, CSS and MySQL.

---

## ✨ Features
- ✅ User Registration and Login
- ✅ Photo and Video Upload
- ✅ Like Posts
- ✅ Comment on Posts
- ✅ Follow / Unfollow Users
- ✅ User Profile Page
- ✅ Live Feed
- ✅ CI/CD Pipeline with Jenkins
- ✅ Docker Containerization
- ✅ MySQL Database
